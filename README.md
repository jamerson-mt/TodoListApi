# 📝 TodoListAPI

Uma Web API robusta e minimalista desenvolvida com **ASP.NET Core** para gestão de tarefas. Este projeto foi concebido com o propósito de consolidar conhecimentos fundamentais em desenvolvimento de APIs, padrões de projeto e integração com banco de dados.

## 🚀 Objetivo do Projeto
O foco principal deste repositório é demonstrar a implementação de um CRUD completo e a fixação de conceitos essenciais:
* **Injeção de Dependência** nativa do .NET.
* Mapeamento de entidades com **Entity Framework Core**.
* Documentação interativa com **Swagger/OpenAPI**.
* Tratamento de erros e validação de modelos (Data Annotations).

## 🛠️ Tecnologias Utilizadas
* **Linguagem:** C#
* **Framework:** ASP.NET Core (Web API)
* **ORM:** Entity Framework Core
* **Banco de Dados:** SQLite / PostgreSQL
* **Documentação:** Swagger

## 📌 Funcionalidades
- [x] **Create:** Criar uma nova tarefa.
- [x] **Read:** Listar todas as tarefas ou buscar por ID.
- [x] **Update:** Atualizar descrição ou status de conclusão.
- [x] **Delete:** Remover tarefas do sistema.
